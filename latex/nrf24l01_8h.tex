This library provides a comprehensive driver for the n\+RF24\+L01 2.\+4\+GHz wireless transceiver module. It supports all major features including multi-\/pipe communication, auto-\/acknowledgment, dynamic payload lengths, and various power/data rate configurations.

\begin{DoxyNote}{Note}
This driver is designed for STM32 microcontrollers using HAL library
