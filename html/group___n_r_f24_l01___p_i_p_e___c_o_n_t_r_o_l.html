Set receive address for specific data pipe.

Parameters
devicePointer to device configuration structure
pipe_numberData pipe number (0-5)
addressPointer to address bytes
lengthAddress length in bytes (3-5)
Returns
0 on success, non-zero on error

Sets the receive address for the specified data pipe.

Note
Pipes 0 and 1 use full address length, pipes 2-5 use only LSB (the remaining bytes are shared with pipe 1).
