nrf24l01_irq irq,
uint8_t enable )

Enable/disable specific interrupt.

Parameters
devicePointer to device configuration structure
irqInterrupt type to control
enable1 to enable, 0 to disable (mask)
Returns
0 on success, non-zero on error

Controls interrupt masking for the specified interrupt source. When enabled (0), the interrupt will trigger the IRQ pin. When disabled (1), the interrupt is masked.

Note
Interrupt enable logic is inverted - 0 enables, 1 disables
